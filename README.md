# 🚀 My PWA App

Progressive Web App (PWA) built with **React** and deployed using **Netlify**.

## ✨ Features

- 📱 Progressive Web App (PWA)
- ⚡ Service Worker
- 📦 Web App Manifest
- 🌐 Offline Support
- 📲 Installable Application
- 🚀 Netlify Deployment

---

## 🛠️ Deployment Guide

### 1️⃣ Fork Repository

Fork repository ini ke akun GitHub Anda.

---

### 2️⃣ Login ke Netlify

Masuk ke:

https://app.netlify.com

Login menggunakan akun **GitHub**.

---

### 3️⃣ Import Repository

Pilih menu berikut:

```
Add new site
        ↓
Import an existing project
        ↓
GitHub
        ↓
Pilih repository hasil Fork
```

---

### 4️⃣ Build Settings

Gunakan konfigurasi berikut.

| Setting            | Value           |
|--------------------|-----------------|
| Build Command      | `npm run build` |
| Publish Directory  | `build`         |

---

### 5️⃣ Deploy

Klik tombol

```
Deploy Site
```

Tunggu hingga proses build selesai.

---

### 6️⃣ Done 🎉

Netlify akan memberikan URL seperti berikut.

```
https://your-project-name.netlify.app
```

Aplikasi siap digunakan sebagai **Progressive Web App**.

---

## 📸 PWA Testing

Buka website yang telah dideploy.

Tekan **F12** kemudian lakukan pengecekan berikut.

- ✅ Application → Manifest
- ✅ Application → Service Workers
- ✅ Lighthouse
- ✅ Install App
